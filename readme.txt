This is the Not Only Team Work Assistant (NOTWA) project repository.

This project has been moved from the merxb/src repo recently while losing all
the history. It was, however, not necessarry as it covered only the early 
phase of the NOTWA project.

The organization file structure is following:

 + Installation                // Installation scripts should be located here
   + Database                  // Installation scripts regarding the Database
     + DatabaseUpgrader        // NetBeans project for the database upgrading

 + NotwaApplication            // The Netbeans project containg the sources
